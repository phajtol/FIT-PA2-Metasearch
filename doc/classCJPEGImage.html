Detailed Description

CJPEGImage is a class that represents JPEG image. CJPEGImage inherits from abstract class CDatabaseEntry. This class contains following metadata: width, height, camera make, camera model and GPS coordinates.

Constructor & Destructor Documentation

◆ CJPEGImage() [1/2]

CJPEGImage::CJPEGImage ( const std::string &  file)

CJPEGImage constructor, calls parent (CDatabaseEntry) constructor and then loads EXIF data from file given as a parameter. CJPEGImage loads image width, height, camera make, camera model and GPS coordinates from file's EXIF data.

Parameters
file- either relative or absolute path to file
